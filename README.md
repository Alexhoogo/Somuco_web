# SOHO外贸产品浏览网站

## 项目简介

这是一个现代化的外贸型产品浏览网站，专为SOHO外贸产品供应商设计。网站采用响应式设计，支持多设备访问，展示各类产品素材。

## 功能特点

### 🎯 核心功能
- **产品轮播展示** - 每个产品分类精选一张图片在首页轮播显示
- **产品分类浏览** - 8个产品分类，每个分类独立页面展示
- **响应式设计** - 完美适配桌面、平板和手机设备
- **现代化UI** - 采用最新的设计趋势和动画效果
- **多语言支持** - 中文界面，适合外贸展示

### 🎨 设计特色
- **渐变背景** - 美观的渐变色彩搭配
- **卡片式布局** - 清晰的产品展示方式
- **悬停动画** - 丰富的交互效果
- **平滑滚动** - 流畅的页面导航体验
- **加载动画** - 优雅的页面加载效果

## 产品分类

网站包含以下8个产品分类：

1. **PU产品** (8款产品)
   - 高品质PU材料，适用于各种应用场景
   
2. **尼龙产品** (4款产品)
   - 耐用尼龙材料，品质保证
   
3. **拉链产品** (21款产品)
   - 精密拉链，品质卓越
   
4. **珠链产品** (20款产品)
   - 精美珠链，时尚设计
   
5. **皮糠纸产品** (11款产品)
   - 环保皮糠纸，绿色选择
   
6. **编织产品** (16款产品)
   - 精美编织，工艺精湛
   
7. **超纤产品** (19款产品)
   - 超细纤维，品质卓越
   
8. **饰品产品** (25款产品)
   - 精美饰品，时尚潮流

## 使用方法

### 本地运行
1. 下载所有文件到本地目录
2. 确保素材文件夹结构正确
3. 用浏览器打开 `index.html` 文件

### 部署到服务器
1. 上传所有文件到Web服务器
2. 确保文件权限正确
3. 访问网站域名

## 自定义说明

### 修改产品信息
编辑各个产品页面的HTML文件，更新产品描述和图片路径。

### 更换图片
将新图片放入对应的素材文件夹，并在HTML中更新图片路径。

### 修改样式
编辑 `styles.css` 文件来自定义颜色、字体、布局等样式。

### 添加新功能
在 `script.js` 文件中添加新的JavaScript功能。
